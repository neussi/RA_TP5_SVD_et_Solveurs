\section{Introduction}
Ce document présente le travail réalisé pour le module sur \textbf{SVD (Singular Value Decomposition)}.
L'objectif est d'implémenter les algorithmes fondamentaux en partant de zéro (Zero STL, C++17).

\section{Concepts Théoriques}
La SVD est le coeur du pipeline AR (Pose, Calibration). Nous avons implementé la methode de Jacobi iterative pour decomposer une matrice A en U*S*V^T, permettant de resoudre des systemes homogènes et de calculer la pseudo-inverse.

\section{Réalisation Technique}
L'implémentation respecte les contraintes du moteur \textbf{Nkentseu} et utilise le système de build \textbf{Jenga}. 
Les points clés incluent :
\begin{itemize}
    \item Decomposition en valeurs singulieres via Jacobi One-Sided.
    \item Utilisation de C++17 sans bibliothèques tierces.
    \item Gestion rigoureuse de la mémoire et de la précision.
\end{itemize}
