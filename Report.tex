\section{Introduction}

Ce rapport presente le module \textbf{SVD et Solveurs Lineaires} du cours
\textit{Realite Augmentee FROM SCRATCH}. La Decomposition en Valeurs Singulieres (SVD)
est au coeur de nombreux algorithmes AR : calibration de camera, calcul d'homographies,
moindres carres.

\section{Concepts Theoriques}

\subsection{Decomposition en Valeurs Singulieres (SVD)}

Toute matrice $A_{m\times n}$ se decompose en :
\[
    A = U \Sigma V^T
\]
ou $U$ et $V$ sont orthogonales et $\Sigma$ est diagonale avec les valeurs singulieres
$\sigma_1 \geq \sigma_2 \geq \ldots \geq 0$.

\textbf{Applications en AR :}
\begin{itemize}
    \item Moindres carres : $\hat{x} = V\Sigma^+U^T b$
    \item Reconstruction approximee : $A \approx \sum_{i=1}^{k} \sigma_i u_i v_i^T$
    \item Estimation de pose (DLT)
\end{itemize}

\subsection{Matrice generique MatXd}

La structure \texttt{MatXd} supporte des matrices de dimensions quelconques :

\begin{lstlisting}[language=C++, caption=Structure MatXd]
struct MatXd {
    int rows, cols;
    std::vector<double> data;
    MatXd(int r, int c) : rows(r), cols(c), data(r*c, 0.0) {}
    double& operator()(int i, int j) { return data[i*cols + j]; }
    MatXd Transpose() const;
};
\end{lstlisting}

\subsection{Algorithme SVD (Golub-Reinsch simplifie)}

L'implementation utilise l'approche iterative Jacobi :
\begin{enumerate}
    \item Bidiagonalisation de Householder
    \item Iterations QR implicites pour diagonaliser
    \item Accumulation des matrices $U$ et $V$
\end{enumerate}
